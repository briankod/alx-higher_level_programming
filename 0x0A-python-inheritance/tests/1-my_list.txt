The 1-my_list module
======================

Using MyList
-------------------

Importing the function from the module:
    >>> MyList = __import__("1-my_list").MyList

Tests whether class MyList inherits from list:
	>>> issubclass(MyList, list)
	True

Tests when unsorted list of integers is given:
    >>> my_list = MyList()
    >>> my_list.append(9)
    >>> my_list.append(8)
    >>> my_list.append(6)
    >>> print(my_list)
    [9, 8, 6]
    >>> my_list.print_sorted()
    [6, 8, 9]


Tests when list is empty:
    >>> my_list = MyList()
    >>> print(my_list)
    []
    >>> my_list.print_sorted()
    []

Tests when list of negative numbers is given:
    >>> my_list = MyList()
    >>> my_list.append(-40)
    >>> my_list.append(-90)
    >>> my_list.append(-1)
    >>> my_list.append(-19)
    >>> print(my_list)
    [-40, -90, -1, -19]
    >>> my_list.print_sorted()
    [-90, -40, -19, -1]

Test when list of integers, some of which are similar:
    >>> my_list = MyList()
    >>> my_list.append(4)
    >>> my_list.append(4)
    >>> my_list.append(6)
    >>> my_list.append(3)
    >>> print(my_list)
    [4, 4, 6, 3]
    >>> my_list.print_sorted()
    [3, 4, 4, 6]
    >>>
