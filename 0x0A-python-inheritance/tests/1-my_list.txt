The 1-my_list module
======================

Using MyList
-------------------

Importing the function from the module:
    >>> MyList = __import__("1-my_list").MyList

Tests when unsorted list of integers is given:
    >>> my_list = MyList()
    >>> my_list.append(9)
    >>> my_list.append(8)
    >>> my_list.append(6)
    >>> print(my_list)
    [9, 8, 6]
    >>> my_list.print_sorted()
    [6, 8, 9]

Tests when list is empty:
    >>> my_list = MyList()
    >>> print(my_list)
    []
    >>> my_list.print_sorted()
    []

Tests when list with some negative numbers is given:
    >>> my_list = MyList()
    >>> my_list.append(-90)
    >>> my_list.append(40)
    >>> my_list.append(19)
    >>> my_list.append(-1)
    >>> print(my_list)
    [-90, 40, 19, -1]
    >>> my_list.print_sorted()
    [-90, -1, 19, 40]

Tests when sorted list(ascending order) is given:
    >>> my_list = MyList()
    >>> my_list.append(2)
    >>> my_list.append(11)
    >>> my_list.append(23)
    >>> print(my_list)
    [2, 11, 23]
    >>> my_list.print_sorted()
